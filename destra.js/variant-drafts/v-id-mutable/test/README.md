# Destra.js 测试文件

本目录包含 Destra.js 的基础功能测试和示例代码。

## 快速开始

### 运行测试

最简单的方式是使用 `tsx`（TypeScript 执行器）：

```bash
# 如果还没有安装 tsx，先安装
npm install -D tsx

# 运行测试
npx tsx test/basic.test.ts
```

或者使用 `ts-node`：

```bash
npx ts-node test/basic.test.ts
```

### 测试内容

`basic.test.ts` 包含了以下测试场景：

1. **基础变量声明和表达式创建** - 展示如何创建变量和使用它们
2. **函数定义和调用** - 展示箭头函数和具名函数的创建与调用
3. **复杂表达式组合** - 展示如何组合多个表达式
4. **方程类型** - 展示显式方程、隐式方程和不等式的创建
5. **ID 设置** - 测试 ID 功能（如果已实现）
6. **依赖关系验证** - 验证表达式之间的依赖关系
7. **错误处理** - 测试错误情况的处理
8. **实际数学场景** - 展示物理和几何场景的实际应用

## 注意事项

- 当前实现支持基础的表达式创建、变量声明和函数定义
- ID 设置和样式 API 等功能可能仍在开发中
- 更多高级功能（如 `selection`、`builder`）请参考项目文档

## 开发提示

如果你想修改或扩展测试：

1. 编辑 `basic.test.ts` 文件
2. 运行测试查看结果
3. 如果遇到类型错误，检查导入的类型是否正确
